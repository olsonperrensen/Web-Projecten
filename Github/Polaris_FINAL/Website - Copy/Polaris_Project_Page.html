HOW POLARIS WAS MADE

Basically it is a combination of what every modern webpage is made out of, and that is HTML for the structural part of it, CSS for the styling part of it, and JavaScript for the interactive `real-time` part of it. I have incorporated an additional tool in order for JavaScript to react to audio more cohesively, and that is a Framework called P5.js
